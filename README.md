# AI Prompt Generator Template

A beautiful, responsive web application for creating customized AI image generation prompts.

## Features

✨ Modern glassmorphism UI  
🎯 5 Categories (Portrait, Landscape, Architecture, Abstract, Product)  
🎨 Customizable styles, moods, and quality levels  
✨ Smart elements selector  
📋 One-click copy to clipboard  
🚀 Quick start with default values  
✅ Pre-flight checklist  
🖨️ Print-friendly design  
📱 Fully responsive  

## Usage

1. Open the website
2. Select a category
3. Customize your prompt settings
4. Click "Copy" to copy the prompt
5. Paste into your favorite AI image generator

## Live Demo

[View Live](https://fatimah-rre.github.io/prompt-generator/)

## Customization

Edit the `CONFIG` object in `index.html` to:
- Change category descriptions
- Modify default values
- Update external links
- Customize colors and styles

## Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- No external dependencies
